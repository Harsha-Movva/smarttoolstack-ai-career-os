# SmartToolStack AI Career OS

AI-powered Career Intelligence Platform built with Next.js, TypeScript, FastAPI, and Python.

## Overview

SmartToolStack AI Career OS helps users analyze resumes, improve ATS performance, identify skill gaps, discover suitable career paths, generate personalized learning roadmaps, and evaluate job-description compatibility.

## Features

* Resume Upload & PDF Parsing
* ATS Score Analysis
* Skill Detection
* Missing Skills Identification
* Career Path Recommendation
* Personalized Learning Roadmap
* Resume Improvement Suggestions
* Job Description Match Analyzer

## Tech Stack
